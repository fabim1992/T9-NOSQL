# Como enviar a resposta

### Pull Requests

1. **Fork** o repositório
2. Crie uma **branch** para sua feature (`git checkout -b feature/matricula-nomeAluno`)
3. **Commit** suas mudanças (`git commit -m 'Adiciona as respostas das questões'`)
4. **Push** para a branch (`git push origin feature/matricula-nomeAluno`)
5. Abra um **Pull Request**
